Changelog
Alpha 0.8.0 current
  • Added a "Reset all Hype" button in Settings to clear the Hype level from every tracked game, with a confirmation first.
  • Renamed the "High Expectation" Hype level to "Idealistic Case" and changed its colour from yellow to green.
Alpha 0.7.0
  • Added a stats panel on wider screens showing totals, next release, and breakdowns by Status, Type, and Hype.
  • The side navigation now shows a count next to each month, "Out now", and "Coming soon".
  • Fixed the side navigation showing as an empty box when no games match the current filters.
  • Fixed the stats panel showing as an empty box when there are no games tracked.
  • The stats panel now hides any category with a count of 0, instead of cluttering the list with zeros.
Alpha 0.6.2
  • Added a new game Type: EAR.
  • Replaced the 🎉 emoji on "Today!" releases with a theme-coloured sparkle icon.
Alpha 0.6.1
  • Removed the "Game" option from Type — leaving Type unset now means the entry is just a regular game.
  • Reordered the filter dropdowns to Type, Status, Hype, matching the add/edit form.
  • Fixed badges overlapping the countdown number in grid view — only 2 badges show, with a "+N" chip for the rest.
  • Standardized filter dropdown capitalization to "All Types", "All Status", "All Hype".
Alpha 0.6.0
  • The side navigation now sits closer to the content with section icons, and hides automatically when there isn't enough room.
  • Added a "Top" shortcut in the side navigation to jump back to the top of the page.
  • Games can now be tagged as Game, DLC, or Patch, shown as a badge on each entry.
  • Added a Hype level (Under the Radar to Terminal Hype) you can set per game, shown as a colour-coded badge.
  • Status, Type, and Hype filters are now compact dropdowns so they fit alongside search on one line.
  • The filter dropdowns now match the selected theme's colours, and a reset button clears all filters and search at once.
  • Added "No Type" and "No Hype" options to their respective filters.
  • The Type field can now be left unset (like Status), so "No Type" actually matches untagged games.
  • The Hype dropdown in the add/edit form now matches the selected theme's colours.
  • Widened the list view for more breathing room.
Alpha 0.5.0
  • A side jump-to navigation appears on wider screens, linking to each month, "Out now", and "Coming soon".
Alpha 0.4.0
  • Search your tracked games by title.
  • Added a "No Status" filter for games with no status set — the Ready filter now only shows games marked Ready.
  • Deleting a game now asks for confirmation first.
  • Released games get their own "Out now" section above "Coming soon", always sorted alphabetically.
  • App updates no longer apply silently — you'll see an "Update now" banner showing the version change, and choose when to refresh.
Alpha 0.3.2
  • Coming soon section sorts alphabetically even with the A-Z sort toggle off.
Alpha 0.3.1
  • Sort the list alphabetically (A-Z) instead of by release date — can be combined with grouping by month.
Alpha 0.3.0
  • The days-remaining countdown glow now matches the selected colour theme.
  • Mark a game "Coming soon" instead of a date — these sink into their own section at the bottom of the list.
  • Set a single status on a game — Ready, Prequel, or Queued — shown as a badge on each entry.
  • Filter the list: All, Ready, Prequel, or Queued only. The Ready filter also includes games with no status set.
  • Import a wishlist from a text file (one title per line) — release dates are looked up automatically on Wikidata.
  • Import: re-search any title to pick a different release-date match, or mark it "Coming soon" if nothing fits.
  • Import: titles you already track are detected, struck through, and skipped automatically.
Alpha 0.2.0
  • Installable as an app on Android Chrome (Add to Home screen).
  • Works offline via a service worker.
  • Added app icons and a fullscreen standalone mode.
  • 5 colour themes (Nebula, Ocean, Sunset, Forest, Midnight) that restyle the whole app.
Alpha 0.1.0
  • Track games with a live days-until-release countdown.
  • Online release-date lookup via Wikidata.
  • Custom card backgrounds: colour swatches or an image URL.
  • Brave image search shortcut for cover art.
  • Add, edit and remove entries.
  • List and grid (cover-art) layouts.
  • Optional grouping by release month.
  • Backup & restore your data.
